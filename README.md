# Wristle
Whistle on your wrist
WatchOS app playing whistle sound maded for P.E teachers
Wristle is an independant watchOS app which is the simplest digital whistle maded for P.E teachers

## Summary


## Requirements
- watchOS 8.0+

## Description
- Whistle
- Timer

## Tech Stack
* Swift
* WatchKit
* Independent Watch App
